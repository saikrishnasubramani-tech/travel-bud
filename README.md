# AI Travel Planner

A beginner-friendly Next.js 15 travel planning homepage built with TypeScript, Tailwind CSS, and the App Router.

## Run locally

Create `.env.local` in the project root:

```bash
GEMINI_API_KEY=your_gemini_api_key_here
```

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.
